# OpenAPI Diff WEB interface
A web interface for an excellent library that detects differences between OpenAPI contracts, available at: https://github.com/OpenAPITools/openapi-diff/
## Build and Run
```bash
mvn clean install
java -jar target/openapi-diff-web-1.0.jar
```
By default, the application runs on port 8080.
